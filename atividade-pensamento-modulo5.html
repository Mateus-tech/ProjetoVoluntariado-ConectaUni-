Módulo 5: Projeto Final

Integre os conhecimentos adquiridos para criar um projeto simples e prático.

Introdução

Neste módulo, você aplicará todos os conceitos aprendidos ao longo do curso para desenvolver um projeto final. O objetivo é criar um sistema simples que resolva um problema cotidiano. Vamos construir um Sistema de Cadastro de Tarefas, onde o usuário pode adicionar, visualizar e remover tarefas.

Problema: Sistema de Cadastro de Tarefas

Imagine que você deseja organizar suas atividades diárias. O sistema deve permitir:

  • Inserir novas tarefas;
  • Listar as tarefas cadastradas;
  • Remover tarefas concluídas.

Pseudocódigo

Início
   Inicialize uma lista vazia de tarefas
   Enquanto o usuário não sair:
       Exiba um menu com as opções: Adicionar, Listar, Remover, Sair
       Se o usuário escolher Adicionar:
           Leia a descrição da tarefa
           Adicione a tarefa à lista
       Se o usuário escolher Listar:
           Exiba todas as tarefas
       Se o usuário escolher Remover:
           Leia o índice ou a descrição da tarefa a remover
           Remova a tarefa da lista
       Se o usuário escolher Sair:
           Termine o programa
   FimEnquanto
Fim

Este pseudocódigo descreve a lógica do sistema: um menu interativo para gerenciar uma lista de tarefas.

Exemplo de Implementação em PHP

<?php
  // Inicia a sessão para armazenar as tarefas
  session_start();
  
  // Se a lista de tarefas não existir, cria um array vazio
  if(!isset($_SESSION['tarefas'])) {
      $_SESSION['tarefas'] = array();
  }
  
  // Se o formulário for enviado, adiciona a nova tarefa
  if(isset($_POST['tarefa'])) {
      $tarefa = trim($_POST['tarefa']);
      if($tarefa != "") {
          array_push($_SESSION['tarefas'], $tarefa);
      }
  }
  
  // Se for solicitado remover uma tarefa
  if(isset($_GET['remover'])) {
      $indice = $_GET['remover'];
      unset($_SESSION['tarefas'][$indice]);
      // Reindexa o array
      $_SESSION['tarefas'] = array_values($_SESSION['tarefas']);
  }
?>
<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>Sistema de Cadastro de Tarefas</title>
</head>
<body>
  <h1>Sistema de Cadastro de Tarefas</h1>
  <form method="post" action="">
      <input type="text" name="tarefa" placeholder="Digite a tarefa" required>
      <button type="submit">Adicionar</button>
  </form>
  <h2>Tarefas Cadastradas:</h2>
  <ul>
  <?php
      foreach($_SESSION['tarefas'] as $indice => $tarefa) {
          echo "<li>$tarefa - <a href='?remover=$indice'>Remover</a></li>";
      }
  ?>
  </ul>
</body>
</html>

Explicação: Este exemplo utiliza sessões para armazenar as tarefas. O usuário insere uma tarefa via formulário; as tarefas são armazenadas em um array na sessão. Um link de remoção permite que tarefas sejam removidas. O código HTML e PHP trabalham juntos para criar um sistema interativo simples.

Exemplo de Implementação em JavaScript (Simulação)

// Exemplo de gerenciamento de tarefas usando JavaScript e armazenamento local (localStorage)

// Função para adicionar uma tarefa
function adicionarTarefa() {
  let tarefa = prompt("Digite a nova tarefa:");
  if(tarefa) {
    let tarefas = JSON.parse(localStorage.getItem("tarefas")) || [];
    tarefas.push(tarefa);
    localStorage.setItem("tarefas", JSON.stringify(tarefas));
    exibirTarefas();
  }
}

// Função para exibir as tarefas
function exibirTarefas() {
  let tarefas = JSON.parse(localStorage.getItem("tarefas")) || [];
  let lista = document.getElementById("listaTarefas");
  lista.innerHTML = "";
  tarefas.forEach((tarefa, index) => {
    lista.innerHTML += `
  • ${tarefa} 
    • `;
  });
}

// Função para remover uma tarefa
function removerTarefa(index) {
  let tarefas = JSON.parse(localStorage.getItem("tarefas")) || [];
  tarefas.splice(index, 1);
  localStorage.setItem("tarefas", JSON.stringify(tarefas));
  exibirTarefas();
}

// Exibe as tarefas ao carregar a página
window.onload = exibirTarefas;

    Explicação: Esse exemplo utiliza o armazenamento local do navegador para salvar as tarefas. O usuário pode adicionar tarefas via prompt e removê-las com um botão. A lista é atualizada dinamicamente.

    Dicas para o Projeto Final

    • Planeje a estrutura do seu projeto usando pseudocódigo ou fluxogramas.
    • Escolha a tecnologia (PHP, JavaScript, etc.) com a qual se sente mais confortável.
    • Implemente funcionalidades de forma incremental e teste cada etapa.
    • Documente o código e mantenha uma boa organização do projeto.
    • Experimente expandir seu projeto adicionando novas funcionalidades aos poucos.

    Conclusão

    O Projeto Final integra os conhecimentos adquiridos ao longo do curso, permitindo que você desenvolva uma solução prática para um problema cotidiano. Ao criar um sistema de cadastro de tarefas, você coloca em prática conceitos de entrada, processamento, armazenamento e exibição de dados – habilidades essenciais para qualquer programador.
