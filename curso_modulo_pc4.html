Curso Módulo 4: Aplicação em Problemas Reais

Transforme a teoria em prática: aprenda a solucionar desafios do dia a dia com programação.

Introdução

Neste módulo, você aplicará os conceitos de algoritmos e programação para resolver problemas reais. Serão apresentados diversos exemplos, desde sistemas simples de gerenciamento até simulações de consultas a APIs, mostrando como transformar ideias em soluções práticas. Cada exemplo inclui pseudocódigo, trechos de código reais e uma explicação detalhada de como o código funciona.

Exemplo 1: Sistema de Gerenciamento de Tarefas

Objetivo: Organizar e priorizar suas tarefas diárias para aumentar a produtividade.

Pseudocódigo:

Início
   Crie uma lista de tarefas
   Para cada tarefa, defina prioridade e prazo
   Ordene as tarefas por prioridade e prazo
   Exiba a lista de tarefas organizada
Fim

Código em JavaScript:

// Array de tarefas com nome e prioridade (1 = alta, 3 = baixa)
let tarefas = [
  { nome: "Estudar programação", prioridade: 1 },
  { nome: "Fazer exercícios", prioridade: 2 },
  { nome: "Reunião com a equipe", prioridade: 1 },
  { nome: "Ler um livro", prioridade: 3 }
];

// Ordena as tarefas pela propriedade 'prioridade'
tarefas.sort((a, b) => a.prioridade - b.prioridade);

// Exibe a lista organizada no console
console.log("Lista de Tarefas Organizada:");
tarefas.forEach(tarefa => console.log("Tarefa:", tarefa.nome, "- Prioridade:", tarefa.prioridade));

Explicação: O código define um array de objetos (tarefas). Utilizamos o método sort() para ordenar as tarefas com base na prioridade e forEach() para exibi-las. Isso mostra como organizar dados de forma lógica.

Exemplo 2: Calculadora de Despesas

Objetivo: Somar várias despesas e obter o total gasto em um período.

Pseudocódigo:

Início
   Crie uma lista com os valores das despesas
   Some todos os valores da lista
   Exiba o total gasto
Fim

Código em Python:

despesas = [120.50, 89.99, 45.00, 300.00]
total = sum(despesas)
print("Total gasto: R$", total)

Explicação: Em Python, a lista despesas armazena valores. A função sum() agrega os valores e o print() exibe o resultado. Esse exemplo ilustra o uso de funções integradas para simplificar operações comuns.

Exemplo 3: Monitoramento de Estoque

Objetivo: Controlar os níveis de produtos em estoque e alertar quando for necessário reabastecer.

Pseudocódigo:

Início
   Para cada produto no estoque:
       Se quantidade < limite mínimo então
           Exiba alerta para reabastecimento
       FimSe
   FimPara
Fim

Código em JavaScript:

let estoque = [
  { produto: "Caneta", quantidade: 50, limite: 20 },
  { produto: "Caderno", quantidade: 15, limite: 10 },
  { produto: "Borracha", quantidade: 5, limite: 8 }
];

estoque.forEach(item => {
  if (item.quantidade < item.limite) {
    console.log("Alerta: Reabasteça o produto", item.produto);
  }
});

Explicação: O código percorre cada item do array estoque e, usando uma condição if, verifica se a quantidade está abaixo do limite. Se estiver, exibe um alerta. Esse exemplo mostra a aplicação de laços e condições para monitorar dados em tempo real.

Exemplo 4: Consulta de Clima Simulada

Objetivo: Simular a consulta a uma API de clima para retornar informações meteorológicas.

Pseudocódigo:

Início
   Faça uma requisição à API de clima (endpoint)
   Receba dados como temperatura e condição do clima
   Exiba os dados para o usuário
Fim

Código em JavaScript (Simulação):

// Função que simula a consulta a uma API de clima
function consultarClima(cidade) {
  // Simulação de resposta da API
  let resposta = { temperatura: 28, condicao: "Ensolarado" };
  return "Em " + cidade + ", a temperatura é de " + resposta.temperatura + "°C e o clima está " + resposta.condicao + ".";
}
let cidade = "São Paulo";
console.log(consultarClima(cidade));

Explicação: A função consultarClima simula uma resposta de API, retornando uma string com os dados meteorológicos. Em um cenário real, utilizaríamos fetch() para obter os dados. Este exemplo reforça o conceito de endpoints e APIs.

Exemplo 5: Carrinho de Compras Simples

Objetivo: Simular um carrinho de compras que adiciona produtos e calcula o total da compra.

Pseudocódigo:

Início
   Crie uma lista de produtos com nome e preço
   Para cada produto adicionado, acumule o valor
   Exiba o total da compra
Fim

Código em JavaScript:

let carrinho = [
  { produto: "Camiseta", preco: 50.00 },
  { produto: "Calça", preco: 120.00 },
  { produto: "Tênis", preco: 200.00 }
];

let totalCompra = 0;
carrinho.forEach(item => {
  totalCompra += item.preco;
});

console.log("Total da compra: R$", totalCompra);

Explicação: Neste exemplo, criamos um array carrinho contendo objetos com os produtos e seus preços. O laço forEach() percorre cada item, somando os preços para obter o total da compra. Essa lógica é comum em sistemas de e-commerce.

Exemplo 6: Sistema de Envio de E-mails Automatizado (Simulação)

Objetivo: Simular um sistema que envia e-mails automáticos para notificações ou confirmações.

Pseudocódigo:

Início
   Para cada usuário na lista:
       Gere uma mensagem personalizada
       Envie o e-mail para o usuário
   FimPara
Fim

Explicação: Embora não possamos enviar e-mails diretamente no navegador sem uma API real, este pseudocódigo mostra a lógica de iterar sobre uma lista de usuários, gerar uma mensagem e enviá-la. Em um ambiente real, você integraria uma API de envio de e-mails (como SendGrid ou Mailgun) para realizar essa tarefa.

Exemplo 7: Análise de Dados Simples

Objetivo: Calcular a média de uma lista de números e identificar o valor máximo e mínimo.

Pseudocódigo:

Início
   Crie uma lista de números
   Calcule a soma de todos os números
   Divida a soma pela quantidade de números para obter a média
   Identifique o valor máximo e o valor mínimo na lista
   Exiba a média, o máximo e o mínimo
Fim

Código em Python:

numeros = [10, 20, 30, 40, 50]
media = sum(numeros) / len(numeros)
valor_max = max(numeros)
valor_min = min(numeros)
print("Média:", media)
print("Valor Máximo:", valor_max)
print("Valor Mínimo:", valor_min)

Explicação: Este código em Python calcula a média de uma lista usando a função sum() e o tamanho da lista (len()). Em seguida, utiliza max() e min() para identificar os valores extremos. Essa abordagem é fundamental para análise de dados básica.

Dicas Gerais para Aplicação em Problemas Reais

  • Divida o problema: Sempre separe o problema em partes menores para facilitar a implementação.
  • Utilize pseudocódigo e fluxogramas: Eles ajudam a organizar suas ideias antes de escrever o código.
  • Teste cada componente: Implemente e teste pequenos protótipos para garantir que cada parte funciona corretamente.
  • Documente seu código: Comentários e documentação facilitam a manutenção e futuras atualizações.

Conclusão

Este módulo demonstrou como os conceitos teóricos podem ser aplicados em problemas reais. Com diversos exemplos – desde gerenciamento de tarefas até análise de dados – você aprendeu a decompor um problema em etapas lógicas, implementar a solução em código e compreender como cada parte funciona.

Pratique esses exemplos, experimente adaptar os códigos às suas necessidades e, com o tempo, você estará preparado para enfrentar desafios mais complexos utilizando programação.
