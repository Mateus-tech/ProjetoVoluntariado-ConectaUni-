Módulo 1: Atividade de Gramática e Vocabulário - Inglês para Dev

Pratique a identificação e o uso de termos técnicos essenciais para desenvolvedores.

Atividades Adicionais: 10 Questões de Gramática e Vocabulário Técnico

A seguir, resolva as 10 questões para praticar a identificação e o uso de termos técnicos e estruturas gramaticais comuns no inglês para desenvolvedores.

Exercício 1: Identificação da Estrutura SVO

Analise a frase: "The programmer debugs the application." Identifique o sujeito, o verbo e o objeto.

  • Sujeito: The programmer
  • Verbo: debugs
  • Objeto: the application

Explicação: A frase segue o padrão SVO, onde o sujeito "The programmer" executa a ação "debugs" sobre o objeto "the application".

Exercício 2: Completar a Frase com o Verbo "to be"

Complete a frase: "The repository ___ updated." Qual é a forma correta do verbo "to be"?

Resposta: is

Explicação: "Repository" é singular, portanto, a forma correta é "is".

Exercício 3: Escolha do Termo Técnico Adequado

Complete a frase: "After the changes, the developer made a ____ to the code." a) merge b) commit c) loop d) API

Resposta: commit

Explicação: "Commit" é o termo usado para registrar alterações no sistema de controle de versão, enquanto "merge" refere-se à união de branches.

Exercício 4: Associação de Termos e Definições

Relacione os termos técnicos com suas definições:

  1. API
  2. Database
  3. Loop
  4. Merge

Definições:

  1. Sistema para armazenar e gerenciar dados.
  2. Conjunto de regras para comunicação entre sistemas.
  3. Repetição de um bloco de código.
  4. Ação de unir alterações de diferentes branches.
  • API: 2
  • Database: 1
  • Loop: 3
  • Merge: 4

Explicação: Cada termo foi associado à definição correta conforme a ordem apresentada.

Exercício 5: Construção de Frase Técnica

Crie uma frase em inglês utilizando pelo menos três dos termos técnicos: API, commit e database. Escreva a frase e explique o significado.

Exemplo de Resposta:

"The developer committed changes after integrating a new API that connects to the database."

Explicação: A frase indica que o desenvolvedor registrou (commit) as alterações feitas após integrar uma nova API, que é responsável por conectar e gerenciar os dados armazenados no database.

Exercício 6: Correção Gramatical

Corrija a frase em inglês para que fique gramaticalmente correta: "Developers uses best practices in code."

Resposta: "Developers use best practices in code."

Explicação: O sujeito "Developers" é plural, portanto o verbo deve estar na forma base "use" em vez de "uses".

Exercício 7: Reescrevendo no Voz Passiva

Reescreva a frase "The team develops a new feature" na voz passiva.

Resposta: "A new feature is developed by the team."

Explicação: Na voz passiva, o objeto da frase original ("a new feature") passa a ser o sujeito, e a ação é expressa com o verbo "is developed" seguido da preposição "by".

Exercício 8: Sinônimos de Termos Técnicos

Forneça um sinônimo para o termo "debug". Explique a importância deste termo no desenvolvimento.

Resposta: "Troubleshoot" pode ser usado como sinônimo de "debug".

Explicação: Debug significa identificar e corrigir erros no código. "Troubleshoot" é o processo de diagnosticar problemas e encontrar soluções, e ambos os termos são essenciais para manter a qualidade e o funcionamento de um software.

Exercício 9: Identificação de Erros em Comentários

Analise o seguinte comentário de código e identifique o erro gramatical: 

  // This function calculates the average of numbers and return the result.
  function calculateAverage(numbers) {
    // código para calcular a média
  }

Resposta: O erro está na frase "and return the result". O verbo deve concordar com "function" ou ser modificado para "and returns the result."

Explicação: Para manter a consistência gramatical, a frase pode ser corrigida para: "This function calculates the average of numbers and returns the result."

Exercício 10: Redação Técnica Curta

Escreva um parágrafo em inglês descrevendo os benefícios de utilizar APIs em projetos de software, utilizando pelo menos dois termos técnicos.

Exemplo de Resposta:

  "Using APIs allows developers to integrate diverse functionalities without reinventing the wheel. It streamlines data management by connecting to a centralized database and ensures efficient communication between different software components."

Explicação: O parágrafo enfatiza como as APIs facilitam a integração de funcionalidades, conectam-se a databases e melhoram a comunicação entre os componentes de um sistema, demonstrando os benefícios técnicos e operacionais dessa abordagem.

Conclusão

Essas 10 atividades foram elaboradas para ajudar você a praticar e consolidar seus conhecimentos em gramática e vocabulário técnico em inglês, essenciais para a comunicação no ambiente de desenvolvimento. Continue praticando e revisando os conceitos para aprimorar sua proficiência e confiança.
