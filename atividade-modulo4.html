Atividade 4: Paradigmas, Funções, Arrays e Manipulação de Dados

Aprenda diferentes paradigmas de programação, como criar e utilizar funções, trabalhar com arrays e manipular dados em PHP.

1. Paradigmas de Programação

PHP suporta diversos paradigmas, permitindo que você escolha o mais adequado para cada situação:

1.1. Paradigma Procedural

No paradigma procedural, o código é organizado em funções que realizam tarefas específicas. Este estilo é simples e direto. 

<?php
  function exibirMensagem($mensagem) {
      echo $mensagem;
  }
  exibirMensagem("Olá, mundo!");
?>

Neste exemplo, a função exibirMensagem() é definida para receber um texto e exibi-lo usando echo.

1.2. Paradigma Funcional

PHP permite o uso de funções anônimas (closures) para trabalhar com o paradigma funcional, que enfatiza funções puras. 

<?php
  $saudacao = function($nome) {
      return "Olá, $nome!";
  };
  echo $saudacao("Carlos");
?>

Aqui, uma função anônima é atribuída à variável $saudacao, que retorna uma saudação personalizada.

1.3. Programação Orientada a Objetos (POO)

A POO organiza o código em objetos, que combinam dados e comportamentos, promovendo modularidade e reutilização. 

<?php
  class Pessoa {
      public $nome;
      public function __construct($nome) {
          $this->nome = $nome;
      }
      public function apresentar() {
          return "Olá, meu nome é " . $this->nome;
      }
  }
  $pessoa = new Pessoa("Mariana");
  echo $pessoa->apresentar();
?>

A classe Pessoa possui uma propriedade e um método para apresentar o objeto. Essa abordagem facilita a manutenção de sistemas complexos.

2. Funções em PHP

2.1. Declaração e Uso de Funções

Funções são blocos de código reutilizáveis que podem receber parâmetros e retornar valores. 

<?php
  function soma($a, $b) {
      return $a + $b;
  }
  echo "Soma: " . soma(10, 20);
?>

A função soma() recebe dois valores, realiza a soma e retorna o resultado.

2.2. Parâmetros com Valor Padrão e Funções Variádicas

Funções podem definir valores padrão para parâmetros e aceitar um número variável de argumentos. 

<?php
  function saudacao($nome = "Visitante") {
      return "Bem-vindo, $nome!";
  }
  echo saudacao();         // Usa valor padrão
  echo saudacao("Mariana");  // Sobrescreve o padrão

  // Função variádica
  function somaTodos(...$numeros) {
      return array_sum($numeros);
  }
  echo somaTodos(1, 2, 3, 4);  // Resultado: 10
?>

O primeiro exemplo usa um valor padrão para o parâmetro. No segundo, o operador ... permite que a função receba vários argumentos.

3. Arrays e Manipulação de Dados

3.1. Tipos de Arrays

Arrays armazenam múltiplos valores. Eles podem ser indexados numericamente ou associativos.

Array Indexado:

<?php
  $frutas = ["Maçã", "Banana", "Laranja"];
  echo $frutas[1]; // Exibe "Banana"
?>

Array Associativo:

<?php
  $pessoa = [
      "nome"  => "João",
      "idade" => 30,
      "cidade" => "São Paulo"
  ];
  echo $pessoa["nome"]; // Exibe "João"
?>

Array Multidimensional:

<?php
  $turmas = [
      "Turma A" => ["Ana", "Bruno", "Carlos"],
      "Turma B" => ["Diana", "Eduardo", "Fernanda"]
  ];
  echo $turmas["Turma A"][1]; // Exibe "Bruno"
?>

Arrays permitem organizar dados de forma estruturada, facilitando a iteração e manipulação com funções como array_sum() e sort().

3.2. Manipulação de Strings e Datas

Strings: Utilize funções como strlen(), strtoupper() e substr() para manipular textos.

<?php
  $texto = "PHP é incrível!";
  echo "Tamanho: " . strlen($texto);
  echo "Em maiúsculas: " . strtoupper($texto);
  echo "Subtexto: " . substr($texto, 0, 3); // Exibe "PHP"
?>

Datas: Funções como date() e strtotime() ajudam a formatar e converter datas.

<?php
  echo "Data atual: " . date("d/m/Y H:i:s");
  $timestamp = strtotime("2025-12-25");
  echo "Timestamp para 25/12/2025: " . $timestamp;
?>

Esses exemplos demonstram a manipulação básica de strings e datas, essenciais para processar informações em aplicações web.

4. Atividades Práticas

Atividade 1: Condicionais Básicas

Crie um script que, com base em uma variável $idade, exiba:

  • "Menor de idade" se $idade for menor que 18;
  • "Adulto" se $idade estiver entre 18 e 64;
  • "Idoso" se $idade for 65 ou maior.
<?php
  $idade = 45;
  
  if ($idade < 18) {
      echo "Menor de idade";
  } elseif ($idade < 65) {
      echo "Adulto";
  } else {
      echo "Idoso";
  }
?>

Explicação: O script utiliza if/elseif/else para definir a faixa etária com base na variável $idade.

Atividade 2: Par ou Ímpar com Ternário

Crie um script que verifique se um número é par ou ímpar utilizando o operador ternário. 

<?php
  $numero = 7;
  $resultado = ($numero % 2 == 0) ? "Par" : "Ímpar";
  echo "O número $numero é $resultado";
?>

Explicação: Se $numero % 2 for igual a zero, o número é par; caso contrário, é ímpar.

Atividade 3: Dia da Semana com Switch

Crie um script que, com base em uma variável $dia (valor de 1 a 7), exiba o dia da semana correspondente utilizando a estrutura switch. 

<?php
  $dia = 4;
  
  switch ($dia) {
      case 1:
          echo "Domingo";
          break;
      case 2:
          echo "Segunda-feira";
          break;
      case 3:
          echo "Terça-feira";
          break;
      case 4:
          echo "Quarta-feira";
          break;
      case 5:
          echo "Quinta-feira";
          break;
      case 6:
          echo "Sexta-feira";
          break;
      case 7:
          echo "Sábado";
          break;
      default:
          echo "Dia inválido";
          break;
  }
?>

Explicação: O switch compara o valor de $dia e exibe o dia correspondente.

Atividade 4: Exibindo Números com While

Crie um script que use um loop while para exibir os números de 1 a 5. 

<?php
  $contador = 1;
  while ($contador <= 5) {
      echo "Número: $contador<br>";
      $contador++;
  }
?>

Explicação: O loop repete enquanto $contador for menor ou igual a 5, incrementando-o a cada iteração.

Atividade 5: Exibindo Números com For

Crie um script que utilize o loop for para exibir os números de 1 a 5. 

<?php
  for ($i = 1; $i <= 5; $i++) {
      echo "Valor de i: $i<br>";
  }
?>

Explicação: O loop for inicia com $i=1 e repete até que $i ultrapasse 5.

Atividade 6: Iterando com Foreach

Crie um script que utilize o loop foreach para percorrer um array de frutas e exibir cada elemento. 

<?php
  $frutas = ["Maçã", "Banana", "Laranja"];
  foreach ($frutas as $fruta) {
      echo "Fruta: $fruta<br>";
  }
?>

Explicação: O loop foreach itera sobre cada elemento do array e exibe o nome de cada fruta.
