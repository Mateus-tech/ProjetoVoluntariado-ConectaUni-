Curso Módulo 3: Estruturas Condicionais e Estruturas de Repetição

Aprenda a controlar o fluxo do seu programa com condições e repetições, fundamentais para qualquer aplicação dinâmica.

1. Estruturas Condicionais

Estruturas condicionais permitem que seu programa tome decisões baseadas em condições que podem ser verdadeiras ou falsas.

1.1. If, Elseif e Else

A estrutura if avalia uma condição; se for verdadeira, executa um bloco de código. O elseif testa condições adicionais e o else executa um bloco se nenhuma das condições anteriores for satisfeita. 

<?php
  $idade = 20;
  
  if ($idade < 18) {
      echo "Menor de idade.";
  } elseif ($idade >= 18 && $idade < 65) {
      echo "Adulto.";
  } else {
      echo "Idoso.";
  }
?>

Neste exemplo, a variável $idade é testada: se for menor que 18, imprime "Menor de idade". Se estiver entre 18 e 64, imprime "Adulto". Caso contrário, imprime "Idoso". A condição ($idade >= 18 && $idade < 65) usa o operador lógico AND (&&) para garantir que ambas as condições sejam verdadeiras.

​

1.2. Operador Ternário

O operador ternário é uma forma resumida de escrever uma condição simples. Ele possui a forma: (condição) ? valor_se_true : valor_se_false; 

<?php
  $numero = 10;
  $resultado = ($numero % 2 == 0) ? "Par" : "Ímpar";
  echo "O número $numero é $resultado.";
?>

Aqui, a expressão $numero % 2 == 0 verifica se o número é divisível por 2 (ou seja, se é par). Se verdadeiro, $resultado recebe "Par"; caso contrário, "Ímpar". Essa estrutura compacta torna o código mais enxuto.

​

1.3. Estrutura Switch

A estrutura switch compara uma variável com vários valores. É útil quando há muitas condições baseadas em um mesmo valor. 

<?php
  $diaSemana = 3;
  
  switch ($diaSemana) {
      case 1:
          echo "Domingo";
          break;
      case 2:
          echo "Segunda-feira";
          break;
      case 3:
          echo "Terça-feira";
          break;
      case 4:
          echo "Quarta-feira";
          break;
      case 5:
          echo "Quinta-feira";
          break;
      case 6:
          echo "Sexta-feira";
          break;
      case 7:
          echo "Sábado";
          break;
      default:
          echo "Dia inválido";
          break;
  }
?>

O switch verifica o valor de $diaSemana e executa o bloco correspondente ao case que casar com esse valor. O comando break garante que o switch saia após a execução do bloco desejado, evitando a execução dos próximos cases.

2. Estruturas de Repetição

Estruturas de repetição permitem executar um bloco de código diversas vezes, seja para repetir uma ação enquanto uma condição for verdadeira ou para percorrer arrays.

2.1. Loop While

O loop while executa seu bloco de código enquanto a condição for verdadeira. 

<?php
  $contador = 1;
  while ($contador <= 5) {
      echo "Contador: $contador<br>";
      $contador++;
  }
?>

Enquanto $contador for menor ou igual a 5, o loop imprime seu valor e incrementa a variável. Quando a condição deixa de ser verdadeira, o loop é interrompido.

2.2. Loop Do...While

O do...while executa o bloco de código primeiro e só depois verifica a condição, garantindo que o código seja executado pelo menos uma vez. 

<?php
  $contador = 1;
  do {
      echo "Contador: $contador<br>";
      $contador++;
  } while ($contador <= 5);
?>

Mesmo que a condição seja falsa logo de início, o bloco dentro do do será executado uma vez antes da verificação.

2.3. Loop For

O loop for é ideal para quando você sabe o número exato de iterações. Ele inclui a inicialização, a condição e o incremento em uma única linha. 

<?php
  for ($i = 1; $i <= 5; $i++) {
      echo "Valor de i: $i<br>";
  }
?>

No exemplo, a variável $i é inicializada com 1, e o loop continua enquanto $i for menor ou igual a 5. Após cada iteração, $i é incrementada em 1.

2.4. Loop Foreach

O foreach é especialmente útil para percorrer arrays. Veja dois exemplos:

Exemplo com array indexado:

<?php
  $frutas = ["Maçã", "Banana", "Laranja"];
  foreach ($frutas as $fruta) {
      echo "Fruta: $fruta<br>";
  }
?>

Exemplo com array associativo:

<?php
  $pessoa = [
      "nome"   => "João",
      "idade"  => 30,
      "cidade" => "São Paulo"
  ];
  foreach ($pessoa as $chave => $valor) {
      echo "$chave: $valor<br>";
  }
?>

No primeiro exemplo, o foreach percorre cada fruta do array e a imprime. No segundo exemplo, o loop percorre cada par chave/valor do array associativo, facilitando a exibição de informações organizadas.

3. Dicas Acadêmicas

Legibilidade: Organize seu código com indentação adequada e comente trechos importantes para facilitar o entendimento.
Testes: Teste cada estrutura individualmente para garantir que o fluxo do programa está correto.
Depuração: Utilize ferramentas como Xdebug para rastrear a execução do código e identificar erros.

4. Conclusão

Dominar as estruturas condicionais e de repetição é essencial para criar lógicas de controle em seus programas. Com elas, você pode definir comportamentos dinâmicos, reagindo a diferentes condições e processando listas de dados de forma eficaz.
