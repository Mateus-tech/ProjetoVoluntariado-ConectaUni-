Atividade 2: Sintaxe e Variáveis em PHP

Aprenda os fundamentos da sintaxe e como declarar e manipular variáveis em PHP.

1. Introdução

Neste módulo, você aprenderá a escrever seu primeiro código PHP, a delimitar os blocos de código e a declarar variáveis, que são os alicerces para a criação de scripts dinâmicos.

2. Conceitos Básicos

2.1. Delimitadores de Código

Todo código PHP deve estar entre <?php e ?>. Essa estrutura informa ao servidor qual parte do arquivo será processada como PHP. 

<?php
  echo "Olá, Mundo!";
?>

O código acima exibe "Olá, Mundo!" no navegador. Apenas o conteúdo entre as tags será interpretado como código PHP.

2.2. Comentários

Comentários são usados para documentar seu código e são ignorados durante a execução. 

// Comentário de uma linha
# Outro comentário de uma linha

/*
  Comentário de múltiplas linhas.
  Útil para explicar trechos complexos.
*/

Os comentários ajudam na manutenção e compreensão do código.

2.3. Variáveis

Variáveis são recipientes para armazenar dados. Em PHP, todas as variáveis começam com o símbolo $. 

<?php
  $mensagem = "Bem-vindo ao PHP!";
  echo $mensagem;
?>

A variável $mensagem armazena o texto e é exibida com echo.

2.4. Regras de Nomeação e Tipagem Dinâmica

Regras importantes:

  • Inicia com $;
  • O primeiro caractere deve ser uma letra ou underline (_);
  • É case sensitive (ex.: $nome ≠ $Nome);
  • Use nomes descritivos.

PHP é de tipagem dinâmica, definindo o tipo conforme o valor atribuído. Você pode converter tipos usando type casting. 

<?php
  $valor = "100";       // $valor é uma string
  $numero = (int)$valor; // Converte para inteiro
  echo $numero;         // Exibe: 100
?>

Converter para inteiro permite operações matemáticas mesmo se o valor original for uma string.

3. Atividades Práticas

Atividade 1: Seu Primeiro Script PHP

Crie um arquivo index.php que contenha:

  • Uso de echo para exibir "Olá, Mundo!"
  • Comentários explicando o que cada linha faz
<?php
  // Exibe uma mensagem de boas-vindas
  echo "Olá, Mundo!";
?>

Explicação: O código inicia com <?php e termina com ?>. A função echo imprime a mensagem "Olá, Mundo!" no navegador.

Atividade 2: Manipulação de Variáveis

Crie um script que:

  • Declare uma variável $nome com seu nome.
  • Declare uma variável $idade com sua idade.
  • Exiba a mensagem: "Olá, meu nome é [nome] e tenho [idade] anos."
<?php
  $nome = "Ana";
  $idade = 25;
  // Exibe a mensagem personalizada
  echo "Olá, meu nome é $nome e tenho $idade anos.";
?>

Explicação: O script declara duas variáveis e utiliza interpolação de strings (com aspas duplas) para exibir os valores.

Atividade 3: Teste de Conversão de Tipos

Crie um script que:

  • Armazene "250" como string em uma variável.
  • Converta essa variável para inteiro.
  • Exiba ambos os valores e informe seus tipos.
<?php
  $valorStr = "250";
  $valorInt = (int)$valorStr;
  
  echo "Valor como string: $valorStr (tipo: " . gettype($valorStr) . ")<br>";
  echo "Valor convertido para inteiro: $valorInt (tipo: " . gettype($valorInt) . ")";
?>

Explicação: A função gettype() mostra o tipo da variável. O type casting converte a string para inteiro.

Atividade 4: Comparando Variáveis

Crie um script que compare duas variáveis numéricas e exiba uma mensagem:

  • Declare $a e $b com valores numéricos.
  • Use if/elseif/else para verificar se $a é maior, menor ou igual a $b.
<?php
  $a = 10;
  $b = 20;
  
  if ($a > $b) {
      echo "$a é maior que $b";
  } elseif ($a < $b) {
      echo "$a é menor que $b";
  } else {
      echo "$a é igual a $b";
  }
?>

Explicação: O script compara os valores e exibe a mensagem correspondente conforme a condição.

Atividade 5: Uso de Comentários e Interpolação

Crie um script que demonstre a diferença entre aspas duplas e simples na interpolação de variáveis. 

<?php
  $nome = "Lucas";
  $frase1 = "Olá, meu nome é $nome!";
  $frase2 = 'Olá, meu nome é $nome!';
  
  echo $frase1; // Exibe: Olá, meu nome é Lucas!
  echo "<br>";
  echo $frase2; // Exibe literalmente: Olá, meu nome é $nome!
?>

Explicação: Aspas duplas interpretam a variável, enquanto aspas simples exibem o texto literalmente.

Atividade 6: Criando um Pequeno Calculador

Crie um script que realize as quatro operações básicas:

  • Declare duas variáveis numéricas, $num1 e $num2.
  • Calcule soma, subtração, multiplicação e divisão.
  • Exiba os resultados com mensagens descritivas.
<?php
  $num1 = 15;
  $num2 = 5;
  
  echo "Soma: " . ($num1 + $num2) . "<br>";
  echo "Subtração: " . ($num1 - $num2) . "<br>";
  echo "Multiplicação: " . ($num1 * $num2) . "<br>";
  echo "Divisão: " . ($num1 / $num2);
?>

Explicação: Cada operação é realizada e o resultado é concatenado com uma string para exibição.

4. Dicas e Recursos Adicionais

  • Tutoriais: Vídeos e artigos online podem ajudar a fixar os conceitos.
  • Prática: Alterar os exemplos e testar diferentes variações é essencial para o aprendizado.
