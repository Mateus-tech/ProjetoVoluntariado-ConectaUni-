Curso Módulo 5: Trabalhando com Formulários e Banco de Dados – Tutorial Passo a Passo

Aprenda a criar formulários, processar dados e integrá-los com um banco de dados MySQL utilizando PDO.

1. Preparando o Ambiente

1.1. Requisitos

Servidor Web: Utilize XAMPP, WAMP ou MAMP para simular um ambiente local com Apache, MySQL e PHP.

Editor de Código: Recomenda-se usar o VS Code para facilitar a edição.

Banco de Dados: MySQL, que geralmente já vem incluso no XAMPP.

Estes requisitos garantem que você terá um ambiente controlado para testar seus scripts PHP e interagir com um banco de dados local.

1.2. Iniciando o Servidor

Abra o painel de controle do XAMPP e inicie os módulos Apache e MySQL.

Crie uma pasta para o seu projeto, por exemplo: C:\xampp\htdocs\meu_projeto.

Dica: Verifique se os serviços estão rodando corretamente antes de prosseguir para evitar problemas de conexão.

2. Criando o Formulário em HTML

Passo 1: Crie um arquivo formulario.html com o código abaixo:

<!DOCTYPE html>
<html lang="pt-BR">
<head>
  <meta charset="UTF-8">
  <title>Cadastro de Usuário</title>
  <style>
    body { font-family: Arial, sans-serif; margin: 20px; }
    form { max-width: 400px; margin: auto; }
    label { display: block; margin-top: 10px; }
    input[type="text"],
    input[type="email"],
    input[type="password"] { width: 100%; padding: 8px; }
    input[type="submit"] { margin-top: 20px; padding: 10px 15px; }
  </style>
</head>
<body>
  <h1>Cadastro de Usuário</h1>
  <form action="processa.php" method="post">
    <label for="nome">Nome:</label>
    <input type="text" id="nome" name="nome" required>
    
    <label for="email">E-mail:</label>
    <input type="email" id="email" name="email" required>
    
    <label for="senha">Senha:</label>
    <input type="password" id="senha" name="senha" required>
    
    <input type="submit" value="Cadastrar">
  </form>
</body>
</html>

Este formulário HTML coleta os dados de nome, e-mail e senha do usuário e envia-os para o arquivo processa.php usando o método POST.

3. Processando os Dados do Formulário com PHP

Passo 2: Crie um arquivo processa.php com o seguinte código:

<?php
  // Captura os dados enviados pelo formulário usando o método POST
  $nome  = trim($_POST['nome'] ?? '');
  $email = trim($_POST['email'] ?? '');
  $senha = trim($_POST['senha'] ?? '');
  
  // Sanitização básica para remover possíveis tags HTML e espaços extras
  $nome  = filter_var($nome, FILTER_SANITIZE_STRING);
  $email = filter_var($email, FILTER_SANITIZE_EMAIL);
  
  // Validação do e-mail para garantir que o formato está correto
  if (!filter_var($email, FILTER_VALIDATE_EMAIL)) {
      die("E-mail inválido!");
  }
  
  // Exibe uma mensagem de confirmação (útil para testes iniciais)
  echo "Bem-vindo, $nome! Seu e-mail é $email.";
?>

Esse script captura os dados enviados, remove espaços e tags indesejadas, valida o e-mail e, em seguida, exibe uma mensagem de boas-vindas. As funções trim() e filter_var() ajudam a manter os dados limpos e seguros.

4. Criando e Configurando o Banco de Dados

Passo 3: No phpMyAdmin, crie um banco de dados chamado meu_banco.

Depois, execute o seguinte comando SQL para criar uma tabela chamada usuarios:

CREATE TABLE usuarios (
    id INT AUTO_INCREMENT PRIMARY KEY,
    nome VARCHAR(100) NOT NULL,
    email VARCHAR(100) NOT NULL UNIQUE,
    senha VARCHAR(255) NOT NULL,
    data_cadastro TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

Este comando SQL cria a tabela usuarios com colunas para ID, nome, e-mail, senha e data de cadastro. A coluna id é auto-incrementada e única, enquanto email deve ser único para evitar duplicatas.

5. Integrando o PHP com o Banco de Dados Usando PDO

Passo 4: Crie um arquivo conexao.php com o código abaixo:

<?php
  $host    = 'localhost';
  $db      = 'meu_banco';
  $user    = 'root';      // Geralmente 'root' no XAMPP
  $pass    = '';          // Normalmente sem senha no XAMPP
  $charset = 'utf8mb4';
  
  $dsn = "mysql:host=$host;dbname=$db;charset=$charset";
  $options = [
      PDO::ATTR_ERRMODE            => PDO::ERRMODE_EXCEPTION,
      PDO::ATTR_DEFAULT_FETCH_MODE => PDO::FETCH_ASSOC,
      PDO::ATTR_EMULATE_PREPARES   => false,
  ];
  
  try {
      $pdo = new PDO($dsn, $user, $pass, $options);
  } catch (PDOException $e) {
      die("Erro na conexão: " . $e->getMessage());
  }
?>

Este script utiliza a extensão PDO para conectar ao banco de dados MySQL. O Data Source Name (dsn) define os parâmetros da conexão, e as opções configuram o tratamento de erros e o modo de busca dos dados.

6. Inserindo os Dados no Banco de Dados

Passo 5: Atualize o arquivo processa.php para inserir os dados na tabela usuarios:

<?php
  require 'conexao.php'; // Inclui o arquivo de conexão
  
  $nome  = trim($_POST['nome'] ?? '');
  $email = trim($_POST['email'] ?? '');
  $senha = trim($_POST['senha'] ?? '');
  
  $nome  = filter_var($nome, FILTER_SANITIZE_STRING);
  $email = filter_var($email, FILTER_SANITIZE_EMAIL);
  
  if (!filter_var($email, FILTER_VALIDATE_EMAIL)) {
      die("E-mail inválido!");
  }
  
  // Criptografa a senha para segurança
  $senhaCriptografada = password_hash($senha, PASSWORD_DEFAULT);
  
  // Prepara a inserção dos dados usando prepared statements
  $stmt = $pdo->prepare("INSERT INTO usuarios (nome, email, senha) VALUES (:nome, :email, :senha)");
  
  try {
      $stmt->execute([
          ':nome'  => $nome,
          ':email' => $email,
          ':senha' => $senhaCriptografada
      ]);
      echo "Usuário cadastrado com sucesso!";
  } catch (PDOException $e) {
      die("Erro ao cadastrar usuário: " . $e->getMessage());
  }
?>

Aqui, os dados capturados são novamente sanitizados e validados. A senha é criptografada com password_hash() para segurança. Em seguida, um prepared statement insere os dados na tabela usuarios. O uso de prepared statements evita ataques de SQL Injection.

7. Dicas, Materiais e Recursos de Estudo

  • Boas Práticas: Sempre valide e sanitize os dados recebidos do usuário.
  • Prepared Statements: Use-os para evitar SQL Injection.
  • Organização: Separe a lógica de conexão, processamento e apresentação em arquivos distintos.
  • Recursos: Consulte a Documentação Oficial do PHP e tutoriais sobre PDO.

8. Conclusão e Próximos Passos

Este tutorial prático o guiou desde a criação de um formulário HTML até a inserção dos dados em um banco de dados MySQL usando PDO. Com esses conhecimentos, você está preparado para desenvolver sistemas web mais complexos, como a implementação de um CRUD completo, autenticação de usuários e muito mais.

Experimente ampliar este exemplo, integrando novas funcionalidades e testando cada etapa para consolidar seu aprendizado.
