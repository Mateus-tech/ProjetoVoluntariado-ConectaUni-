Curso Módulo 2: Sintaxe e Variáveis em PHP

Aprenda em detalhes como funciona a sintaxe do PHP, além de trabalhar com variáveis, constantes e tipos de dados de maneira simples e prática.

1. Introdução à Sintaxe do PHP

O PHP é uma linguagem de script usada no lado do servidor. Isso significa que ele processa o código no servidor e gera o HTML que será enviado para o navegador. Nesta seção, vamos explicar como iniciar e encerrar um bloco PHP e qual a importância de manter a estrutura correta.

Como funciona: O código PHP deve estar contido entre as tags <?php e ?>. Tudo que estiver dentro dessas tags será interpretado pelo servidor. Código fora dessas tags é considerado HTML e será enviado diretamente para o navegador.

2. Blocos de Código e Comentários

a) Blocos de Código

Todo script PHP inicia com <?php e termina com ?>. Isso informa ao servidor que o código entre essas tags deve ser interpretado como PHP. 

<?php
  echo "Olá, Mundo!";
?>

No exemplo acima, a função echo envia a mensagem "Olá, Mundo!" para o navegador. O código entre as tags é processado pelo PHP, e o resultado final é um HTML com a mensagem.

b) Comentários

Comentários são trechos do código que são ignorados pelo interpretador, servindo apenas para documentar o que o código faz. 

// Este é um comentário de uma linha
# Também é um comentário de uma linha

/*
  Este é um comentário de múltiplas linhas.
  Pode ser usado para explicar trechos complexos.
*/

Ao utilizar comentários, você torna seu código mais fácil de entender, tanto para você quanto para outras pessoas que possam trabalhar nele futuramente.

3. Variáveis em PHP

a) O que são variáveis?

Variáveis são "caixas" que armazenam informações que podem ser usadas e modificadas durante a execução do seu programa. Em PHP, todas as variáveis começam com o símbolo $. 

<?php
  $nome = "Maria";  // Armazena uma string com o nome
  $idade = 25;      // Armazena um número inteiro
  echo "Olá, $nome! Você tem $idade anos.";
?>

Neste exemplo, $nome guarda o valor "Maria" e $idade guarda o número 25. A função echo exibe a mensagem completa, substituindo as variáveis pelos seus respectivos valores.

b) Regras para Nomeação

Para criar uma variável válida:

  • Ela deve começar com o cifrão $.
  • O primeiro caractere após o $ deve ser uma letra ou underscore (_), nunca um número.
  • Os nomes são sensíveis a maiúsculas e minúsculas ($nome é diferente de $Nome).
  • Use nomes descritivos para facilitar o entendimento do código.

Dica: Pense em variáveis como rótulos para informações. Escolha nomes que deixem claro o que está armazenado, como $idadeUsuario ou $precoProduto.

c) Tipagem Dinâmica e Conversão

PHP é uma linguagem de tipagem dinâmica, o que significa que você não precisa definir o tipo da variável explicitamente. O tipo é determinado pelo valor atribuído. 

<?php
  $valor = "100";       // Aqui, $valor é uma string
  $numero = (int)$valor; // Converte a string para inteiro
  echo $numero;         // Exibe: 100
?>

No exemplo, a variável $valor contém o texto "100". Usando o type casting (int), convertemos esse valor para o número 100, permitindo operações matemáticas.

4. Constantes

Constantes são valores imutáveis, ou seja, uma vez definidas, não podem ser alteradas durante a execução do programa. 

<?php
  define("VERSAO_PHP", "8.3"); // Define uma constante usando define()
  const DIRETORIO = "/var/www/php"; // Outra forma de definir constante
  echo "Estamos usando PHP " . VERSAO_PHP;
?>

O exemplo define duas constantes: VERSAO_PHP e DIRETORIO. Elas servem para armazenar informações que não devem mudar, como a versão do PHP ou o caminho do diretório.

5. Arrays e Outros Tipos de Dados

a) Arrays Simples e Associativos

Arrays são estruturas que armazenam vários valores em uma única variável. Eles podem ser indexados numericamente ou por chaves (arrays associativos). 

<?php
  // Array indexado
  $cores = array("vermelho", "verde", "azul");
  
  // Array associativo
  $pessoa = array(
    "nome" => "Carlos",
    "idade" => 30,
    "cidade" => "Feira de Santana"
  );
  
  echo $cores[0];       // Exibe: vermelho
  echo $pessoa["nome"];   // Exibe: Carlos
?>

No primeiro array, os valores são armazenados em posições numéricas (0, 1, 2, ...). No segundo, cada valor tem uma chave descritiva (como "nome" ou "idade"), facilitando o acesso aos dados.

b) Outros Tipos de Dados

  • Inteiros: Números sem decimais, como 25.
  • Float: Números com decimais, como 3.14.
  • Strings: Texto, por exemplo, "Olá".
  • Booleanos: Valores true (verdadeiro) ou false (falso).
  • Null: Representa a ausência de valor.

6. Exemplos Práticos

Exemplo 1: Mensagem Personalizada

<?php
  $usuario = "João";
  $mensagem = "Bem-vindo, $usuario! Hoje é um ótimo dia para aprender PHP.";
  echo $mensagem;
?>

Este código armazena o nome "João" na variável $usuario e monta uma mensagem personalizada usando essa variável. Quando echo é executado, a mensagem completa é exibida.

Exemplo 2: Operações com Variáveis

<?php
  $a = 5;
  $b = 3;
  $soma = $a + $b;      // Realiza a soma de 5 + 3
  $produto = $a * $b;   // Realiza a multiplicação de 5 * 3
  
  echo "Soma: " . $soma . "<br>";
  echo "Produto: " . $produto;
?>

Aqui, definimos duas variáveis numéricas e, em seguida, calculamos a soma e o produto. O operador + soma os valores, enquanto o * multiplica. O resultado de cada operação é mostrado na tela.

Exemplo 3: Trabalhando com Arrays e Looping

<?php
  $frutas = array("Maçã", "Banana", "Laranja");
  foreach($frutas as $fruta) {
    echo "Eu gosto de " . $fruta . "<br>";
  }
?>

O loop foreach percorre cada elemento do array $frutas e, para cada item, exibe uma mensagem. Essa estrutura é muito útil para trabalhar com listas de dados.

7. Boas Práticas e Dicas Extras

  • Mantenha nomes de variáveis e constantes descritivos.
  • Comente seu código para facilitar o entendimento.
  • Teste cada trecho separadamente para identificar e corrigir erros.
  • Utilize identação e espaçamento consistentes para melhorar a legibilidade.
