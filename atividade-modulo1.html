Atividade 1: Instalação e Configuração do PHP

Configuração do ambiente e primeiros scripts

1. Introdução

Neste módulo, você aprenderá a instalar o PHP, configurar seu ambiente de desenvolvimento e criar seus primeiros scripts.

2. Instalação do PHP

2.1. Download e Extração

Acesse o site oficial do PHP (php.net/downloads) e baixe a versão "Thread Safe" adequada ao seu sistema. Extraia o arquivo para uma pasta (ex.: C:\php).

A versão "Thread Safe" é recomendada para uso com servidores web como o Apache. A extração do ZIP cria a pasta onde os arquivos PHP serão armazenados.

2.2. Configuração das Variáveis de Ambiente

Para que o PHP seja reconhecido pelo sistema, adicione o caminho da pasta (ex.: C:\php) à variável de ambiente Path do Windows.

Isso permite que você execute o comando php -v no Prompt de Comando para verificar a versão instalada.

3. Criando seu Primeiro Script PHP

3.1. Arquivo Index.php

Crie um arquivo chamado index.php na pasta do projeto com o seguinte conteúdo: 

<?php
  echo "Olá, Mundo!";
?>

Esse script básico utiliza a função echo para exibir a mensagem "Olá, Mundo!" no navegador. Salve o arquivo e execute-o via servidor local.

4. Atividades para Praticar

Atividade 1: Verificar Instalação

Abra o Prompt de Comando ou Terminal e digite: php -v. Registre a versão do PHP instalada e confirme se a instalação foi bem-sucedida.

Atividade 2: Criação de Script Simples

Crie um arquivo index.php que:

  • Exiba a mensagem "Bem-vindo ao PHP!" usando echo.
  • Adicione um comentário explicando o que o script faz.

Atividade 3: Configuração do Ambiente Local

Instale o XAMPP (ou outro pacote similar), inicie os módulos Apache e MySQL e crie uma pasta para seu projeto em C:\xampp\htdocs. Acesse o seu script index.php no navegador via http://localhost/nomedapasta e observe o resultado.

5. Dicas e Recursos Adicionais

  • Documentação Oficial: Consulte php.net/manual para mais detalhes sobre instalação e configuração.
  • Tutoriais: Procure por vídeos e tutoriais online que demonstrem a configuração do ambiente local com XAMPP.
  • Comunidades: Participe de fóruns e grupos de discussão sobre PHP para trocar dúvidas e experiências.

6. Conclusão

Com este módulo, você aprendeu a instalar o PHP, configurar as variáveis de ambiente e criar seu primeiro script. As atividades propostas ajudarão a consolidar o conhecimento adquirido.
