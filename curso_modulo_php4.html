Curso Módulo 4: Paradigmas, Funções, Arrays, Manipulação de Dados e POO em PHP

Aprenda conceitos fundamentais como os paradigmas de programação, declaração e uso de funções, manipulação de arrays e a Programação Orientada a Objetos (POO).

1. Paradigmas de Programação em PHP

1.1. Paradigma Procedural

No paradigma procedural, o código é organizado em funções que realizam tarefas específicas. Essa abordagem é simples e ideal para scripts pequenos. 

<?php
  function exibirMensagem($mensagem) {
      echo $mensagem;
  }
  // Chama a função passando uma mensagem
  exibirMensagem("Olá, mundo!");
?>

Aqui, definimos uma função exibirMensagem() que recebe um parâmetro e utiliza echo para imprimir a mensagem. Em seguida, chamamos a função com o argumento "Olá, mundo!". Esse é o básico do estilo procedural, onde o foco é em funções que manipulam dados.

1.2. Paradigma Funcional

PHP também suporta funções anônimas e closures, aproximando-o do paradigma funcional. Essa abordagem enfatiza funções puras – sem efeitos colaterais – e o uso de funções como parâmetros. 

<?php
  $saudacao = function($nome) {
      return "Olá, $nome!";
  };
  // Chamada da função anônima armazenada em $saudacao
  echo $saudacao("Carlos");
?>

Neste exemplo, uma função anônima é atribuída à variável $saudacao. Ao chamar $saudacao("Carlos"), a função retorna a saudação para "Carlos". Esse estilo permite passar funções como argumentos e utilizar técnicas funcionais como map, filter e reduce.

1.3. Programação Orientada a Objetos (POO)

A POO organiza o código em objetos, que combinam dados e comportamentos. Essa abordagem facilita a modularidade e a manutenção de sistemas complexos.

2. Funções em PHP

2.1. Declaração e Uso de Funções

Funções são blocos de código reutilizáveis que podem receber parâmetros e retornar valores. Elas ajudam a evitar a repetição de código. 

<?php
  function soma($a, $b) {
      return $a + $b;
  }
  $resultado = soma(10, 20);
  echo "A soma é: $resultado";
?>

A função soma() recebe dois números, soma-os e retorna o resultado. Depois, armazenamos o resultado em $resultado e o exibimos.

2.2. Parâmetros com Valor Padrão e Variádicos

É possível definir valores padrão para parâmetros, que serão usados caso nenhum argumento seja passado. Funções variádicas aceitam um número indefinido de argumentos.

Exemplo com Valor Padrão:

<?php
  function saudacao($nome = "Visitante") {
      return "Bem-vindo, $nome!";
  }
  echo saudacao();         // Usa o valor padrão "Visitante"
  echo saudacao("Mariana"); // Sobrescreve o valor padrão
?>

Exemplo com Variádicos:

<?php
  function somaTodos(...$numeros) {
      return array_sum($numeros);
  }
  echo somaTodos(1, 2, 3, 4);  // Resultado: 10
?>

No primeiro exemplo, se nenhum nome for passado para saudacao(), o padrão "Visitante" é usado. No segundo, o operador ... permite passar vários números, que são somados com a função array_sum().

2.3. Funções Anônimas e Recursividade

Função Anônima:

<?php
  $quadrado = function($numero) {
      return $numero * $numero;
  };
  echo "O quadrado de 5 é " . $quadrado(5);
?>

Função Recursiva:

<?php
  function fatorial($n) {
      if ($n <= 1) {
          return 1;
      } else {
          return $n * fatorial($n - 1);
      }
  }
  echo "Fatorial de 5 é: " . fatorial(5);
?>

A função anônima $quadrado calcula o quadrado de um número. Já a função fatorial() chama a si mesma para calcular o fatorial, demonstrando recursividade – onde uma função resolve um problema chamando a si mesma até atingir uma condição de parada.

3. Arrays e Manipulação de Dados

3.1. Tipos de Arrays

Arrays Indexados: Armazenam valores usando índices numéricos.

<?php
  $frutas = ["Maçã", "Banana", "Laranja"];
  echo $frutas[1]; // Exibe "Banana"
?>

Arrays Associativos: Utilizam chaves descritivas para acessar valores.

<?php
  $pessoa = [
      "nome"  => "João",
      "idade" => 30,
      "cidade" => "Rio de Janeiro"
  ];
  echo $pessoa["nome"]; // Exibe "João"
?>

Arrays Multidimensionais: Arrays dentro de arrays, úteis para organizar dados complexos.

<?php
  $turmas = [
      "Turma A" => ["Ana", "Bruno", "Carlos"],
      "Turma B" => ["Diana", "Eduardo", "Fernanda"]
  ];
  echo $turmas["Turma A"][1]; // Exibe "Bruno"
?>

3.2. Funções para Manipulação de Arrays

PHP oferece várias funções para inserir, remover, mesclar e ordenar arrays, além de aplicar funções como map, filter e reduce.

Exemplo - Inserir e Remover:

<?php
  $numeros = [1, 2, 3];
  array_push($numeros, 4); // Adiciona 4 no final
  $removido = array_pop($numeros); // Remove o último elemento (4)
?>

Exemplo - Mesclar e Ordenar:

<?php
  $a = [3, 1, 4];
  sort($a); // Ordena o array
  print_r($a); // Exibe [1, 3, 4]
?>

Exemplo - Map, Filter e Reduce:

<?php
  $numeros = [1, 2, 3, 4, 5];
  $quadrados = array_map(function($n) {
      return $n * $n;
  }, $numeros);
  print_r($quadrados);
?>

Estes exemplos demonstram como adicionar e remover elementos de um array, ordenar e aplicar transformações (como calcular o quadrado de cada número). Funções como array_map() facilitam o processamento de listas.

3.3. Manipulação de Strings e Datas

Strings: Funções como strlen(), strtoupper() e substr() ajudam a trabalhar com textos.

<?php
  $texto = "PHP é incrível!";
  echo "Tamanho: " . strlen($texto);
  echo "Em maiúsculas: " . strtoupper($texto);
  echo "Subtexto: " . substr($texto, 0, 3); // Exibe "PHP"
?>

Datas: As funções date() e strtotime() são usadas para formatar e converter datas.

<?php
  echo "Data atual: " . date("d/m/Y H:i:s");
  $timestamp = strtotime("2025-12-25");
  echo "Timestamp para 25/12/2025: " . $timestamp;
?>

3.4. Exemplo Integrado: Processamento de Alunos

<?php
  // Função para calcular a média de notas
  function calcularMedia(array $notas) {
      return array_sum($notas) / count($notas);
  }
  
  // Array de alunos com suas notas
  $alunos = [
      ["nome" => "Mariana", "notas" => [8.5, 7.0, 9.0]],
      ["nome" => "Rafael", "notas" => [6.5, 7.5, 8.0]],
      ["nome" => "Carla", "notas" => [9.0, 8.5, 9.5]]
  ];
  
  // Filtrar alunos aprovados (média >= 7.0) e incluir a média calculada
  $aprovados = array_filter($alunos, function($aluno) {
      $media = calcularMedia($aluno["notas"]);
      $aluno["media"] = $media;
      return $media >= 7.0;
  });
  
  // Exibir os alunos aprovados com a média formatada
  foreach ($aprovados as $aluno) {
      echo "Aluno: {$aluno['nome']} - Média: " . number_format($aluno['media'], 2) . "<br>";
  }
?>

Este exemplo mostra como calcular a média das notas de cada aluno, filtrar aqueles com média maior ou igual a 7.0 e exibir os resultados formatados. É uma aplicação prática que integra funções, arrays e manipulação de dados.

4. Programação Orientada a Objetos (POO) em PHP – Detalhamento

4.1. Classes e Objetos

Em POO, classes são modelos para criar objetos. Um objeto é uma instância de uma classe, contendo propriedades (dados) e métodos (comportamentos). 

<?php
  class Pessoa {
      public $nome;
      public $idade;
  
      // Construtor: executado quando o objeto é criado
      public function __construct($nome, $idade) {
          $this->nome  = $nome;
          $this->idade = $idade;
      }
  
      // Método para apresentar a pessoa
      public function apresentar() {
          return "Olá, meu nome é {$this->nome} e tenho {$this->idade} anos.";
      }
  }
  
  $pessoa1 = new Pessoa("Mariana", 28);
  echo $pessoa1->apresentar();
?>

A classe Pessoa define propriedades e métodos. O construtor inicializa as propriedades quando um objeto é criado. O método apresentar() retorna uma string com os dados da pessoa.

4.2. Encapsulamento e Modificadores de Acesso

O encapsulamento restringe o acesso direto às propriedades de uma classe. Os modificadores de acesso (public, private, protected) definem essa visibilidade. 

<?php
  class ContaBancaria {
      private $saldo = 0; // Propriedade privada, só acessível dentro da classe
  
      public function depositar($valor) {
          if ($valor > 0) {
              $this->saldo += $valor;
          }
      }
  
      public function getSaldo() {
          return $this->saldo;
      }
  }
  
  $conta = new ContaBancaria();
  $conta->depositar(100);
  echo "Saldo: " . $conta->getSaldo();
?>

Aqui, o atributo $saldo é privado e só pode ser modificado pelos métodos públicos depositar() e getSaldo(). Essa técnica protege os dados contra alterações indesejadas.

4.3. Herança e Polimorfismo

A herança permite que uma classe herde propriedades e métodos de outra. O polimorfismo possibilita que métodos sejam redefinidos em classes filhas para comportamentos específicos. 

<?php
  class Animal {
      public $nome;
  
      public function __construct($nome) {
          $this->nome = $nome;
      }
  
      public function emitirSom() {
          return "Som genérico";
      }
  }
  
  class Cachorro extends Animal {
      public function emitirSom() {
          return "Au Au";
      }
  }
  
  $cachorro = new Cachorro("Buddy");
  echo "{$cachorro->nome} diz: " . $cachorro->emitirSom();
?>

A classe Animal define um método genérico. A classe Cachorro estende Animal e redefine o método emitirSom() para retornar "Au Au". Assim, o mesmo método se comporta de forma diferente conforme a classe.

4.4. Abstração, Interfaces e Traits

Classe Abstrata:

<?php
  abstract class Forma {
      // Método abstrato que deve ser implementado pelas classes filhas
      abstract public function area();
  }
  
  class Retangulo extends Forma {
      private $largura;
      private $altura;
  
      public function __construct($largura, $altura) {
          $this->largura = $largura;
          $this->altura  = $altura;
      }
  
      public function area() {
          return $this->largura * $this->altura;
      }
  }
  
  $retangulo = new Retangulo(10, 5);
  echo "Área do retângulo: " . $retangulo->area();
?>

Interface:

<?php
  interface Logavel {
      public function log($mensagem);
  }
  
  class Sistema implements Logavel {
      public function log($mensagem) {
          echo "Log: " . $mensagem;
      }
  }
  
  $sistema = new Sistema();
  $sistema->log("Sistema iniciado.");
?>

Trait:

<?php
  trait Logger {
      public function log($mensagem) {
          echo "Log: " . $mensagem . "<br>";
      }
  }
  
  class Usuario {
      use Logger;
      private $nome;
  
      public function __construct($nome) {
          $this->nome = $nome;
      }
  }
  
  class Produto {
      use Logger;
      private $descricao;
  
      public function __construct($descricao) {
          $this->descricao = $descricao;
      }
  }
  
  $usuario = new Usuario("Ana");
  $produto = new Produto("Camiseta");
  
  $usuario->log("Usuário logado.");
  $produto->log("Produto adicionado ao carrinho.");
?>

As classes abstratas servem de modelo para as filhas. Interfaces definem contratos que as classes devem cumprir. Traits permitem reutilizar métodos entre classes sem herança. Essas técnicas aumentam a modularidade e a reutilização do código.

4.5. Exemplo Prático Integrado de POO

<?php
  // Classe abstrata Animal
  abstract class Animal {
      protected $nome;
  
      public function __construct($nome) {
          $this->nome = $nome;
      }
  
      abstract public function emitirSom();
  }
  
  // Classe Cachorro que estende Animal
  class Cachorro extends Animal {
      public function emitirSom() {
          return "Au Au";
      }
  }
  
  // Classe Gato que estende Animal
  class Gato extends Animal {
      public function emitirSom() {
          return "Miau";
      }
  }
  
  // Criação de um array com diferentes animais
  $animais = [
      new Cachorro("Rex"),
      new Gato("Mimi")
  ];
  
  foreach ($animais as $animal) {
      echo "{$animal->nome} diz: " . $animal->emitirSom() . "<br>";
  }
?>

Este exemplo integra herança, polimorfismo e abstração. Criamos uma classe abstrata Animal com um método abstrato emitirSom() que é implementado de maneira diferente pelas classes Cachorro e Gato. Assim, ao percorrer o array de animais, o método correto é chamado para cada objeto.
