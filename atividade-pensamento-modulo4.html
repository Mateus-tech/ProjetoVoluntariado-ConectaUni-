Módulo 4: Aplicação Real

Desenvolva uma solução para um problema cotidiano utilizando lógica computacional.

Introdução

Neste módulo, você vai aplicar seus conhecimentos de lógica computacional para desenvolver uma solução prática para um problema real do dia a dia. A ideia é que você identifique o problema, estruture um algoritmo para resolvê-lo e veja uma implementação simples que pode ser adaptada conforme necessário.

Problema: Controle de Despesas Mensais

Imagine que você deseja controlar suas despesas mensais para entender melhor seus gastos. O objetivo é criar um algoritmo que:

  • Permita ao usuário inserir diversos valores de despesas.
  • Calcule o total gasto no mês.
  • Calcule a média das despesas.
  • Exiba uma análise simples dos gastos.

Pseudocódigo:

Início
   Inicialize uma lista vazia para despesas
   Solicite ao usuário o número de despesas a registrar
   Para cada despesa:
       Leia o valor da despesa
       Adicione o valor à lista
   FimPara
   Calcule o total somando todos os valores da lista
   Calcule a média dividindo o total pelo número de despesas
   Exiba o total e a média
Fim

O pseudocódigo acima descreve os passos básicos: coletar os dados, processá-los (somar e calcular a média) e exibir os resultados.

Exemplo de Implementação em JavaScript:

// Solicita ao usuário a quantidade de despesas
let qtdDespesas = parseInt(prompt("Quantas despesas você deseja registrar?"));

// Cria um array para armazenar os valores das despesas
let despesas = [];

// Coleta cada despesa
for (let i = 0; i < qtdDespesas; i++) {
  let valor = parseFloat(prompt("Digite o valor da despesa " + (i + 1) + ":"));
  despesas.push(valor);
}

// Calcula o total de despesas
let total = despesas.reduce((acumulador, valorAtual) => acumulador + valorAtual, 0);

// Calcula a média das despesas
let media = total / qtdDespesas;

// Exibe os resultados
alert("Total gasto no mês: R$ " + total.toFixed(2) + "\nMédia das despesas: R$ " + media.toFixed(2));

Explicação: O código solicita ao usuário o número de despesas e, em seguida, utiliza um laço for para coletar cada valor, armazenando-os em um array. A função reduce() é usada para somar todos os valores, e a média é calculada dividindo o total pelo número de despesas. Por fim, os resultados são exibidos em um alert.

Exemplo de Implementação em Python:

# Solicita o número de despesas
qtd_despesas = int(input("Quantas despesas você deseja registrar? "))

# Inicializa uma lista para armazenar as despesas
despesas = []

# Coleta os valores das despesas
for i in range(qtd_despesas):
    valor = float(input(f"Digite o valor da despesa {i+1}: "))
    despesas.append(valor)

# Calcula o total e a média
total = sum(despesas)
media = total / qtd_despesas

# Exibe os resultados
print(f"Total gasto no mês: R$ {total:.2f}")
print(f"Média das despesas: R$ {media:.2f}")

Explicação: O script em Python pede ao usuário o número de despesas e, em seguida, utiliza um loop for para coletar cada valor. A função sum() calcula o total e a média é obtida dividindo o total pelo número de despesas. Os resultados são então impressos na tela.

Dicas para Aplicar Lógica em Problemas Reais

  • Analise o problema e divida-o em partes menores.
  • Utilize pseudocódigo ou fluxogramas para planejar a solução.
  • Implemente e teste cada componente separadamente.
  • Documente o processo para facilitar futuras manutenções.

Conclusão

Aplicar a lógica computacional para resolver problemas cotidianos é fundamental para transformar ideias em soluções práticas. Este módulo demonstrou como criar um sistema simples para controlar despesas mensais, mas a abordagem pode ser adaptada para diversos outros desafios. Continue praticando e explorando novas maneiras de aplicar o pensamento computacional em situações reais.
