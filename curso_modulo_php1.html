Curso Módulo 1: Instalação e Configuração do PHP no VS Code

Aprenda Instalação e Configuração do PHP no VS Code

1. Introdução ao PHP

O que é PHP?

PHP (Hypertext Preprocessor) é uma linguagem de script server-side usada para desenvolvimento web.

  • Execução no Servidor: Processa os scripts e gera HTML dinâmico para o cliente.
  • Integração com HTML e Banco de Dados: Facilita a criação de páginas web dinâmicas e a interação com bancos como MySQL.
  • Comunidade e Ferramentas: Suporta grandes sistemas (WordPress, Laravel, etc.) e conta com vasta documentação e comunidade ativa.

Por que aprender PHP?

  • Fácil Integração: Mistura com HTML e conexão com bancos de dados de maneira simples.
  • Ampla Aplicação: Utilizado em diversos CMS e frameworks, facilitando o ingresso no mercado de trabalho.
  • Recursos e Suporte: Grande quantidade de tutoriais, cursos e bibliotecas.

2. Instalação do PHP

A instalação do PHP varia conforme o sistema operacional. Confira os passos para Windows, macOS e Linux.

2.1. Windows

Download: Acesse php.net/downloads e escolha a versão “Thread Safe” (ex: PHP 8.3.x VS16 x64).

Extração: Extraia o arquivo ZIP para o diretório C:\php.

Configuração das Variáveis de Ambiente: Pressione Win + S e procure por “variáveis de ambiente”. Em “Variáveis do Sistema”, edite a variável Path e adicione o caminho C:\php.

Verificação: Abra o Prompt de Comando e execute:

php -v

A saída deverá indicar a versão instalada (ex: PHP 8.3.x (cli)...).

2.2. macOS

Instalação do Homebrew (se necessário):

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

Instalar PHP via Homebrew:

brew install php

Verificar a instalação:

php -v

2.3. Linux (Ubuntu/Debian)

Atualize os repositórios:

sudo apt update

Instale o PHP:

sudo apt install php

Verificação:

php -v

2.4. Instalação de Extensões (Opcional)

Windows: Edite o arquivo php.ini (localizado em C:\php) e descomente as linhas necessárias (ex.: extension=gd).

Linux/macOS:

sudo apt install php-mysql php-gd php-curl

3. Configuração do VS Code para PHP

3.1. Instalação das Extensões

No VS Code, instale as seguintes extensões essenciais:

  • PHP Intelephense: Fornece autocompletar, análise estática e documentação integrada.
  • PHP Debug: Permite depuração do código PHP com Xdebug.
  • Live Server (opcional): Atualiza automaticamente a visualização do navegador ao salvar arquivos.

3.2. Configuração do Servidor Local

Para testar seus arquivos PHP, utilize um servidor local. Duas opções comuns:

  • XAMPP/MAMP: Servidores pré-configurados ideais para iniciantes.
  • Servidor embutido do PHP: Execute o comando abaixo no terminal na pasta do projeto:
php -S localhost:8000

3.3. Configuração da Depuração com Xdebug

1. Instale o Xdebug (siga as instruções do wizard do Xdebug).

2. Configure o php.ini adicionando:

zend_extension="caminho/para/xdebug.so"  # (.dll no Windows)
xdebug.mode=debug
xdebug.start_with_request=yes

3. No VS Code, pressione F5 e selecione “PHP”. Um arquivo launch.json será gerado, contendo:

{
  "version": "0.2.0",
  "configurations": [
    {
      "name": "Listen for Xdebug",
      "type": "php",
      "request": "launch",
      "port": 9003
    }
  ]
}

4. Criando o Primeiro Arquivo PHP

1. Crie uma pasta para o projeto (ex: meu_site) e abra-a no VS Code.

2. Crie o arquivo index.php com o seguinte conteúdo:

<?php
echo "Olá, mundo!";
?>

3. No terminal, execute:

php -S localhost:8000

Acesse http://localhost:8000 e visualize a mensagem “Olá, mundo!”.

5. Exemplo Prático: Página Dinâmica

Crie o arquivo exemplo.php com o seguinte conteúdo:

<!DOCTYPE html>
<html>
  <head>
    <meta charset="UTF-8">
    <title>Página Dinâmica</title>
  </head>
  <body>
    <?php
      $nome = "João";
      echo "<h1>Bem-vindo, $nome!</h1>";
    ?>
  </body>
</html>

6. Troubleshooting Comum

Problema: PHP não reconhecido no terminal
Solução: Verifique se o caminho do PHP está corretamente definido nas variáveis de ambiente.

Problema: Erros de Sintaxe
Solução: Assegure-se de abrir e fechar o bloco PHP corretamente com <?php ... ?> e utilizar pontos-e-vírgula (;) ao final das instruções.

Problema: Extensões não carregadas
Solução: Edite o arquivo php.ini para habilitar as extensões necessárias e reinicie o servidor.
