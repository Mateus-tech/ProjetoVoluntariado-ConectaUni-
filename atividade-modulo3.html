Atividade 3: Estruturas Condicionais e Estruturas de Repetição

Aprenda a controlar o fluxo do seu programa com condições e loops.

1. Introdução

Neste módulo, você aprenderá como utilizar estruturas condicionais para tomar decisões e estruturas de repetição para executar tarefas repetidamente. Estes conceitos são fundamentais para desenvolver lógica em seus scripts.

2. Estruturas Condicionais

2.1. If, Elseif e Else

A estrutura if executa um bloco de código se uma condição for verdadeira. Com elseif você pode testar outras condições, e else executa um bloco se nenhuma das condições anteriores for satisfeita. 

<?php
  $idade = 20;
  
  if ($idade < 18) {
      echo "Menor de idade";
  } elseif ($idade < 65) {
      echo "Adulto";
  } else {
      echo "Idoso";
  }
?>

Este exemplo verifica o valor da variável $idade e exibe uma mensagem apropriada conforme a faixa etária.

2.2. Operador Ternário

O operador ternário é uma forma compacta de escrever uma condição simples. Sua sintaxe é: (condição) ? valor_se_true : valor_se_false. 

<?php
  $numero = 10;
  $resultado = ($numero % 2 == 0) ? "Par" : "Ímpar";
  echo "O número $numero é $resultado";
?>

Se a condição ($numero % 2 == 0) for verdadeira, $resultado receberá "Par"; caso contrário, "Ímpar".

2.3. Estrutura Switch

A estrutura switch compara o valor de uma variável com vários casos, tornando o código mais legível quando há várias condições. 

<?php
  $dia = 3;
  
  switch ($dia) {
      case 1:
          echo "Domingo";
          break;
      case 2:
          echo "Segunda-feira";
          break;
      case 3:
          echo "Terça-feira";
          break;
      case 4:
          echo "Quarta-feira";
          break;
      case 5:
          echo "Quinta-feira";
          break;
      case 6:
          echo "Sexta-feira";
          break;
      case 7:
          echo "Sábado";
          break;
      default:
          echo "Dia inválido";
          break;
  }
?>

O switch verifica o valor de $dia e exibe o dia correspondente. O comando break encerra a execução após o caso correspondente.

3. Estruturas de Repetição

3.1. Loop While

O loop while executa um bloco de código enquanto a condição for verdadeira. 

<?php
  $contador = 1;
  while ($contador <= 5) {
      echo "Número: $contador<br>";
      $contador++;
  }
?>

Enquanto $contador for menor ou igual a 5, o loop repete e incrementa o valor, exibindo os números de 1 a 5.

3.2. Loop Do...While

O loop do...while executa o bloco de código pelo menos uma vez antes de verificar a condição. 

<?php
  $contador = 1;
  do {
      echo "Número: $contador<br>";
      $contador++;
  } while ($contador <= 5);
?>

Mesmo se a condição não for atendida inicialmente, o bloco dentro do do será executado pelo menos uma vez.

3.3. Loop For

O loop for é ideal quando o número de iterações é conhecido, integrando a inicialização, condição e incremento. 

<?php
  for ($i = 1; $i <= 5; $i++) {
      echo "Valor de i: $i<br>";
  }
?>

O loop inicia com $i = 1 e repete até que $i seja maior que 5, incrementando $i a cada iteração.

3.4. Loop Foreach

O foreach é utilizado para percorrer arrays, facilitando a iteração sobre seus elementos.

Exemplo com Array Indexado:

<?php
  $frutas = ["Maçã", "Banana", "Laranja"];
  foreach ($frutas as $fruta) {
      echo "Fruta: $fruta<br>";
  }
?>

Exemplo com Array Associativo:

<?php
  $pessoa = [
      "nome" => "João",
      "idade" => 30,
      "cidade" => "São Paulo"
  ];
  foreach ($pessoa as $chave => $valor) {
      echo "$chave: $valor<br>";
  }
?>

O foreach percorre cada elemento do array, exibindo as informações de forma simples e organizada.

4. Atividades Práticas

Atividade 1: Condicionais Básicas

Crie um script que, baseado em uma variável $idade, exiba:

  • "Menor de idade" se $idade for menor que 18;
  • "Adulto" se $idade estiver entre 18 e 64;
  • "Idoso" se $idade for 65 ou maior.
<?php
  $idade = 45;
  
  if ($idade < 18) {
      echo "Menor de idade";
  } elseif ($idade < 65) {
      echo "Adulto";
  } else {
      echo "Idoso";
  }
?>

Explicação: O script utiliza if/elseif/else para determinar a faixa etária com base no valor de $idade.

Atividade 2: Verificação Par ou Ímpar (Operador Ternário)

Crie um script que verifique se um número é par ou ímpar utilizando o operador ternário e exiba a mensagem. 

<?php
  $numero = 7;
  $resultado = ($numero % 2 == 0) ? "Par" : "Ímpar";
  echo "O número $numero é $resultado";
?>

Explicação: Se a condição ($numero % 2 == 0) for verdadeira, o número é par; caso contrário, ímpar.

Atividade 3: Exibindo o Dia da Semana com Switch

Crie um script que utilize a estrutura switch para exibir o dia da semana, baseado em uma variável $dia com valor de 1 a 7. 

<?php
  $dia = 5;
  
  switch ($dia) {
      case 1:
          echo "Domingo";
          break;
      case 2:
          echo "Segunda-feira";
          break;
      case 3:
          echo "Terça-feira";
          break;
      case 4:
          echo "Quarta-feira";
          break;
      case 5:
          echo "Quinta-feira";
          break;
      case 6:
          echo "Sexta-feira";
          break;
      case 7:
          echo "Sábado";
          break;
      default:
          echo "Dia inválido";
          break;
  }
?>

Explicação: O switch compara o valor de $dia e exibe o dia correspondente. O break garante que apenas o caso correto seja executado.

Atividade 4: Loop While

Crie um script que use o loop while para exibir os números de 1 a 5. 

<?php
  $contador = 1;
  while ($contador <= 5) {
      echo "Número: $contador<br>";
      $contador++;
  }
?>

Explicação: Enquanto $contador for menor ou igual a 5, o loop repete e exibe o número, incrementando-o a cada iteração.

Atividade 5: Loop Do...While

Crie um script que utilize o loop do...while para exibir os números de 1 a 5, garantindo que o bloco seja executado pelo menos uma vez. 

<?php
  $contador = 1;
  do {
      echo "Número: $contador<br>";
      $contador++;
  } while ($contador <= 5);
?>

Explicação: O bloco de código é executado pelo menos uma vez, mesmo que a condição seja falsa inicialmente.

Atividade 6: Loop For

Crie um script que utilize o loop for para exibir os números de 1 a 5. 

<?php
  for ($i = 1; $i <= 5; $i++) {
      echo "Valor de i: $i<br>";
  }
?>

Explicação: O loop inicia com $i = 1 e repete até que $i seja maior que 5, incrementando $i a cada iteração.

Atividade 7: Loop Foreach

Crie um script que utilize o loop foreach para percorrer um array de frutas e exibir cada elemento. 

<?php
  $frutas = ["Maçã", "Banana", "Laranja"];
  foreach ($frutas as $fruta) {
      echo "Fruta: $fruta<br>";
  }
?>

Explicação: O foreach itera sobre cada elemento do array, facilitando a exibição dos dados.
